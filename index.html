Work

  1. Water Treatment Plant Data Analysis
    • Demonstrated experience in utilizing machine learning methods to optimize operations at a Water Treatment Plant (WTP).
    • Utilized statistical methods such as Pearson correlation coefficient and Spearman rank correlation coefficient to find the correlation between different attributes.
    • Visualized correlations using scatter plots and clustered water samples based on their similar characteristics using K-Means and Hierarchical Clustering methods.
    • Implemented classification methods like decision trees, random forests, neural networks, and support vector machines (SVM) to predict which water samples are healthy or hazardous.
  2. Distributed Movie Ticket Booking System
    • Designed and developed a Distributed Java application for booking of Movie tickets on three different theatre servers Java, Java RMI and User Datagram Protocol(UDP)
    • There were three different servers created assumed for three theatres. The client and server connection was implemented using Java RMI.
    • Server to Server connectivity was established using UDP.
    • The Admin user could create movie slots, remove movie slots and view the movie slots available on all servers. The customer could book a movie ticket, cancel a movie ticket and also view the movie tickets he/she booked accross all servers.
  3. Flight Ticket Booking Software
    • Built a user friendly software to book tickets. The flight data was downloaded using the Skyscanner API and database operations were performed using PostgreSQL.
  4. Covid 19 Fact Checker - Project Demo Video
    • Developed a user-friendly web-based application to detect whether a news related to Covid is fake or real. The application was developed using machine learning training model Multinomial Naïve Bayes and was deployed on the web using Amazon Web Services EC2.
  5. Built A ChatBot For College Library Website
    • Developed a chatbot for my college library.
    • The chatbot was used to handle library-related questions from college students.
    • The chatbot was built using google cloud technologies Dialogflow and Firebase.
  6. IntelliSupport ChatBot - OrganicBots Internship Project
    • Built a chatbot for a water filter company.
    • The chatbot acted as a customer service support for the users where the customers could share the issue being faced and schedule for an executive visit to resolve the issue whenever required.
    • The chatbot was built using google cloud technologies Dialogflow and Firebase.

About

I am currently a Master’s student at Concordia University in Montreal. My degrees major is Applied Computer Science. I began my degree in the fall of 2022 and will finish it in the summer of 2024. My first semester of study is already complete where I had taken the courses Programming and Problem Solving & Advance Programming Practices. Currently in my second semester I am studying Distributed System Design & Big Data Analytics. I have 1.5 years of experience working as a FrontEnd developer for India’s largest IT firm, ’Tata Consultancy Services,’ on a banking project for the client State Bank Of India. I got to work on a variety of coding languages here, including VB.NET, Java, HTML, CSS, and Java. I also had the opportunity to work with government APIs that had to be integrated with the banking application, which required a significant amount of cryptography to secure the input and output requests.

In addition to my professional experience, I also have experience as a Research Analyst Intern at Organic Bots, Incubator Center. During my time there from January 2018 to January 2019, I worked on researching and building small applications using UiPath, Google DialogFlow, and Machine Learning classifiers. The applications that I built included a ChatBot using Google DialogFlow, UiPath Application for text scraping automation for application form filling, and Fake News Detection using Naive Bayes Algorithm.

I’ve always been a valuable member of whichever team I’ve been a part of. During my undergraduate studies, I was elected as the Head of the College’s Technical Fest, and during my first year at the company, I was named Employee of the Year. I am a great team player who is always eager to learn new things
